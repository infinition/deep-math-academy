Le Determinant : L Ame de la Matrice

Une matrice transforme l'espace. Elle peut l'etirer, le tourner, ou l'ecraser.
Le Determinant est un nombre unique qui mesure comment le volume (ou l'aire) change.

Transformation de l'Espace (2D)

Interactif
Deplace les vecteurs i (Rouge) et j (Vert)
Matrice A :
1.0 0.0 0.0 1.0
Determinant (Aire)
1.00
Inversible

Interpretation Geometrique

Le determinant est le facteur d'echelle de l'aire.
Si det = 2, l'aire double.
Si det = 0.5, l'aire est divisee par 2.

Matrice Singuliere (Det = 0)

Si les vecteurs sont alignes (colineaires), l'aire devient nulle.
L'espace 2D est ecrase sur une ligne 1D.
Consequence : On ne peut pas inverser la matrice (on ne peut pas "desecraser" une ligne pour refaire un plan).

Signe Negatif ? Si le determinant est negatif, l'espace a ete retourne (comme un miroir).
