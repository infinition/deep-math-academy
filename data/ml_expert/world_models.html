VLM / VLA / World Models

Ce chapitre relie perception multimodale, action et planification en espace latent.

Equation de dynamique latente

$$z_{t+1}=f_\theta(z_t,a_t),\quad \hat{o}_{t+1}=g_\theta(z_{t+1})$$

Analogie: simulateur mental interne qui teste des actions avant execution reelle.

VLM / MLLM

Fusion texte + image + parfois audio/video pour raisonner sur des scenes.

$$s=\mathrm{sim}(E_{text}(t),E_{img}(i))$$

VLA (Vision-Language-Action)

Le modele observe, comprend l instruction et emet une action.

$$a_t\sim\pi_\theta(a\mid o_t,\text{instruction})$$

World Models en RL/robotique

Dreamer-like, latent planning, model-based RL, diffusion world models.

$$\pi^*=\arg\max_\pi\;\mathbb{E}\left[\sum_t r_t\right] \text{ via planification sur } z_t$$

RAG multimodal + agents

Recherche d evidences texte/image/tableaux + appels outils + verification.

$$\text{Decision}=f(\text{Reasoning},\text{Retrieved Evidence},\text{Tool Outputs})$$

Applications concretes

  • Robot mobile: navigation et manipulation avec camera + langage
  • Document AI: comprehension de tableaux/charts + extraction fiable
  • Assistant industriel: planification d actions securisees
  • Agent logiciel: lecture interface + action outillee + memoire
